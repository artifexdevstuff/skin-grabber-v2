# Privacy Policy for Skin Grabber V2

*Last Updated: June 13, 2026*

This Privacy Policy explains how Skin Grabber V2 ("the Bot") handles information. We value your privacy and aim to be as transparent as possible.

### 1. Data We Collect
**The Bot does not store any personal user data.** 
- We do **not** have a database.
- We do **not** store Discord User IDs, server IDs, or message content.
- We do **not** log your command history.

### 2. How We Process Information
When you use the `/skin` command, the Bot temporarily processes the Minecraft username you provide to:
- Communicate with the Mojang API to find the player's UUID.
- Request a render from Crafatar or Minotar.
This data is processed in real-time and is not saved after the response is sent to Discord.

### 3. Third-Party Services
To provide its service, the Bot interacts with the following third-party APIs:
- **Mojang API**: To convert usernames to UUIDs.
- **Crafatar & Minotar**: To generate skin renders.
Please refer to the respective privacy policies of these services for information on how they handle data.

### 4. Data Retention
Since the Bot is stateless and has no database, we retain **zero** user data. Once a command is executed, the data is gone from the Bot's memory.

### 5. Your Rights
Because we do not store any of your data, there is no data for you to access, rectify, or delete. If you wish to stop using the Bot, you may simply remove it from your Discord server.

### 6. Contact Information
If you have any questions regarding this Privacy Policy, please contact the developer via the official support channels or the GitHub repository.

*By using this Bot, you acknowledge that you have read and understood this Privacy Policy.*
